# パターン選択ガイド(使うべきタイミング一覧)

ユーザーの依頼文・内容から最適なレイアウトパターンを選ぶための判断基準。
**上から順に判定し、最初に条件が合致したパターンを使う。** 迷ったら②ビジネス型。

## 判定フロー

0. **写真・ビジュアルを全面に使いたい**か?(画像提供あり/世界観重視の表紙) → ㉟画像ヒーロー型。**製品画面・UIを見せたい**なら61プロダクトUIショーケース型
1. 資料の**冒頭/末尾の定型パート**か? → 目次なら⑫アジェンダ型、章の切り替わりなら㊱セクション区切り型、まとめなら㉝まとめ型、行動喚起なら㉚CTA型、**会議の締め・議事共有**(決定/宿題)なら㊾決定事項・アクション型
2. 依頼に**時系列・順序**があるか? → 手順なら⑥フロー型、四半期/年単位の計画なら⑧ロードマップ型、過去の歩みなら⑱沿革型、詳細工程表なら㉗ガントチャート型、循環するなら㉕サイクル型、イベント当日の時刻付きタイムテーブルなら㊸プログラム型、**導入後・入社後の30-60-90日**なら52プラン型、**年間の施策一覧**なら53年間カレンダー型、**顧客体験の段階ごとの感情・タッチポイント**なら57カスタマージャーニー型、**資金調達の実績・ラウンド推移**なら62資金調達ラウンド型
3. **数値データ**が主役か? → 推移なら⑩グラフ型、単発実績なら④KPI型、複数指標一覧なら㉘ダッシュボード型、市場規模なら㉑TAM/SAM/SOM型、転換率なら㉓ファネル型、**内訳・構成比**なら㊴構成比型、**アンケート・調査結果**なら㊵アンケート型、**目標に対する進捗**なら㊶OKR型、**組織・文化を数字の一覧で紹介**(採用の「数字で見る◯◯」)なら㊻数字で見る型、**投資対効果・回収期間**なら51 ROI型、**売上・利益の増減要因分解**(ブリッジ/ウォーターフォール)なら63売上ブリッジ型
4. **対比・分類**が主題か? → Before/Afterなら⑨対比型、現状→目標+施策なら㉔ギャップ分析型、2軸整理なら⑪マトリクス型、多社の機能比較なら㉒競合比較表型、階層なら㉖ピラミッド型、強み/弱み/機会/脅威なら㊷SWOT分析型、**複数案から推奨を決める**なら㊲意思決定型、**リスクと対策を対で示す**なら㊿リスク&対策型
5. **構造・体制**の説明か? → 組織/体制なら⑭組織図型、技術構成なら⑮システム構成図型、サービス全体像なら⑳ソリューション型、**お金と価値の流れ**(ビジネスモデル)なら㊽ビジネスモデル相関図型、**海外拠点・グローバル展開**なら54拠点型、**周辺ツール・パートナーとの連携**(エコシステム)なら55エコシステム型
6. **人・実績・信頼**の訴求か? → 1人の深掘り紹介(登壇者・講師・自己紹介)なら㊼プロフィール型、**狙う顧客像の深掘り**(ペルソナ)なら56ペルソナ型、複数メンバーなら⑰チーム紹介型、顧客の声なら⑬お客様の声型、1社の事例深掘りなら㊳ケーススタディ型、取引先実績なら㉜ロゴウォール型、**受賞歴・メディア掲載実績**なら58受賞・メディア掲載型、料金なら⑯料金プラン型、FAQなら㉛FAQ型
7. **並列の項目**(特徴・プラン)か? → 3〜4項目なら③3カード型。**種類の違う情報**(価値+KPI+機能など)を1枚で俯瞰させたいなら㉞ベントーグリッド型、**募集職種の一覧**なら59求人ポジション一覧型、**福利厚生・社内制度の紹介**なら60福利厚生型
8. **メッセージ1つ**を届けたいか? → 課題の問いかけなら⑲課題提起型、名言/理念の言葉なら㉙引用型、「〜とは?」の概念説明なら㊺用語解説型、表紙/ビジョンなら①コンセプト型(和風なら⑦和モダン型)
9. **告知・お知らせ**か? → 新機能・リリースなら㊹アップデート型、toC・カジュアルな告知なら⑤カジュアル型
10. 上記以外(要点説明・提案の骨子) → ②ビジネス型

## 組み合わせのヒント

- 各実例ファイルの冒頭コメントにも同じ使用タイミングが書いてある。ベースに流用するときはコメントごと更新する
- どのパターンでも**配色は内容のトーンに合わせて差し替えてよい**(パターン=レイアウト構造の選択であり、色は自由)
- ユーザーが「画像全面」を求めた場合は㉟画像ヒーロー型(slide-hero)をベースに、`.bg-placeholder` を `<img>`(object-fit: cover)に差し替える。オーバーレイ層は必ず残す
